# AUBG_Hack
AUBG Hack 3.0 Project

# Pre-install tools needed:
- JDK 11 or higher
- Maven
- Docker

# Start:
cd backend \
mvn clean package \
cd .. \
docker-compose up --build -d

# Hint:
You may need to convert the wait-for-it.sh file type to Unix through a text editor.

# Running on:
http://localhost:8080
